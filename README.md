# Лабораторна робота: Інтерактивні фазові портрети для систем диференціальних рівнянь

## Опис

Цей застосунок дозволяє досліджувати фазові портрети двовимірних динамічних систем. Ви можете:
- Вибирати одну з двох
 систем (або додати власну)
- Візуалізувати векторне поле, ізокліни (nullclines) та струминні лінії (streamlines)
- Досліджувати типи особливих точок (рівноваг)
- Інтерактивно додавати початкові точки для інтегрування траєкторій
- Змінювати область, щільність поля, тривалість інтегрування та інші параметри
- Зберігати графік у PNG

## Як запустити

1. Встановіть залежності:
    ```
    pip install dash plotly numpy scipy
    ```

2. Запустіть файл:
    ```
    python main.py
    ```

3. Відкрийте браузер і перейдіть за адресою http://127.0.0.1:8050/

## Структура застосунку

- **Вибір системи:** Dropdown у верхній частині сторінки
- **Візуалізація:** Графік з векторним полем, ізоклінами, струминними лініями та особливими точками
- **Параметри області:** Зміна меж, щільності, тривалості інтегрування
- **Початкові точки:** Додаються кліком по графіку, можна очистити
- **Таблиця власних значень:** Для кожної особливої точки показано λ1, λ2 та тип

## Приклад систем

- **System A:** dx/dt = -y, dy/dt = x - 3x²
- **System B:** dx/dt = -2xy, dy/dt = x² + y² - 1

## Можливості

- Темна/світла тема
- Збереження графіка
- Інтерактивна робота з фазовим портретом
